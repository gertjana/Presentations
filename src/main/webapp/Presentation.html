    browser.type("id=loginForm:password", "demo");
    browser.click("id=loginForm:login")
    browser.waitForPageToLoad("15000");

    Assert.assertTrue("User should be logged in!",
        browser.isElementPresent("xpath=//li[contains(text(), 'Welcome')]"));

new Java API's (couple of talks combined)

  • JSR 353: Java API for JSON Parser
  • Stream based, like the StAX Parser
    register to Events to handle objects as they come in
    Will probably in JavaEE 7
  • JSR 356: Java API for websocket
  • Simple protocol to send/receive text/binary, some async support works by opening a standard http connection and then upgrade to websocket
    will be in JavaEE 7

Lambda's, (parallel) collections

Java (JDK 8)
(String s1, String s2) -> s1.length() - s2.length();
Scala (now)
        (s1: String, s2: String) => s1.length - s2.length

Ceylon, Kotlin, Clojure have similar constructions
