TomEE - a Java EE6 Application Server based on Tomcat

  • Standard Tomcat with added enterprise features
    (EJB, JPA, CDI, JSF, WS-*)
  • Passes the EE6 Certification tests
  • Fast and Lean and still looks and acts like Tomcat
  • Convention over configuration:
    The minimum configuration needed is: 
    <tomee></tomee>

new Java API's (couple of talks combined)

  • JSR 353: Java API for JSON Parser
  • Stream based, like the StAX Parser
    register to Events to handle objects as they come in
    Probably in JavaEE 7
  • JSR 356: Java API for websocket
  • Simple protocol to send/receive text/binary, some async support works by opening a standard http connection and then upgrade to websocket
    will be in JavaEE 7

Effective Scala

Use the for loop to validate parameters or to filter collections

scala 
def userPermissions(session:Session, username:String, password:String) {
    for {
        u < username;
        p < password;
    } yield session.getPermissions(u,p);
}
